# 口头对战游戏 - 游戏设计文档

## 游戏概述
回合制多人对战游戏，支持 2-8 名玩家，每位玩家选择一个角色进行战斗，直到只剩一名存活者。

## 核心机制

### 角色属性
| 属性 | 说明 |
|------|------|
| 生命值 (HP) | 为0时角色死亡 |
| 魔法值 (MP) | 释放技能消耗 |
| 敏捷值 (AGI) | 决定回合行动顺序 |
| 韧性值 (DEF) | 减免伤害程度 |
| 怒气值 (RAGE) | 受到伤害时累积，部分技能消耗 |

### 回合动作
1. **普通攻击** - 造成基础伤害
2. **防御** - 本回合韧性+50%
3. **逃跑** - 50%概率躲避攻击（特定技能可提升至70%）
4. **使用技能** - 消耗MP/RAGE释放特殊效果

### 伤害公式
```
最终伤害 = 基础伤害 × 条件加成 × (10 - 韧性值) / 10
```
- 数值精确到小数点后两位

### 回合流程
1. **动作选择阶段** - 所有玩家选择本回合动作
2. **动作结算阶段** - 按敏捷值从高到低依次结算
3. **死亡检查阶段** - 检查并移除HP≤0的角色

## 角色设计

### 骑士 (Knight)
**定位**: 均衡型战士，技能多样

| 属性 | 数值 |
|------|------|
| 生命值 | 300 |
| 魔法值 | 150 |
| 敏捷值 | 7 |
| 韧性值 | 2 |
| 普攻伤害 | 30 |

**技能列表**:
| 技能 | 消耗 | 效果 |
|------|------|------|
| 升龙击 | 10 MP | 对目标造成50点伤害 |
| 冲锋 | 10 HP | 对目标造成75点伤害，目标逃跑则无效 |
| 追砍 | 10 MP | 对目标造成15点伤害，目标逃跑则×4 |
| 太阳之弓 | 15 MP | 对目标造成30点伤害，目标用技能则×2 |
| 马鞭 | 无 | 敏捷+1，逃跑成功率提升至70% |
| 马踏飞燕 | 80 RAGE | 对目标造成50点伤害，目标防御则×1.5 |

| 属性 | 数值 |

| 敏捷值 | 4 |
| 韧性值 | 3 |

| 重锤 | 20 MP | 对目标造成50点伤害，目标未逃跑则×1.5 |

## 技术架构

### 文件结构
```
Game/
├── src/
│   ├── core/           # 游戏引擎核心
│   │   ├── GameEngine.js
│   │   ├── BattleSystem.js
│   │   └── DamageCalculator.js
│   ├── characters/     # 角色定义
│   │   ├── Character.js
│   │   ├── Knight.js
│   │   ├── ArmoredWarrior.js
│   │   └── Archer.js
│   ├── skills/         # 技能系统
│   │   └── SkillManager.js
│   ├── ui/             # 用户界面
│   │   ├── GameUI.js
│   │   └── AnimationManager.js
│   └── assets/         # 资源文件
│       ├── images/
│       ├── animations/
│       └── sounds/
├── docs/               # 文档
├── tests/              # 测试
└── index.html          # 入口文件
```
