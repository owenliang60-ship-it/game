> 测试日期: 2026-02-03

## 一、角色数据总览

### 1.1 基础属性对比

| 属性 | 骑士 | 装甲战士 | 弓箭手 |
|------|------|---------|--------|
| HP | 300 | 350 | 270 |
| MP | 150 | 120 | 150 |
| AGI | 7 | 4 | 6 (→8游击后) |
| DEF | 2 | 3 (→5顶盾后) | 1.5 |
| ATK | 30 | 25 | 20 |
| 被动 | 无 | 钢铁意志(+5怒气/回合) | 游击战术+精确瞄准 |

### 1.2 有效生命值 (EHP) 分析

```
EHP = HP / (1 - DEF/10)

骑士:    300 / 0.8 = 375 有效HP
装甲战士: 350 / 0.7 = 500 有效HP
弓箭手:   270 / 0.85 = 317.6 有效HP

顶盾后装甲战士: 350 / 0.5 = 700 有效HP
```

**结论**: 装甲战士EHP是弓箭手的1.57倍，顶盾后达到2.2倍。

## 三、模拟战斗场景

### 场景1: 开局3回合 (无特殊策略)

```
=== 回合1 ===
行动顺序: 骑士(AGI7) → 弓箭手(AGI6) → 装甲战士(AGI4)
装甲战士回合开始: +5怒气 (累计5)

假设动作:
- 骑士: 升龙击 → 弓箭手
- 弓箭手: 太阳之弓 → 装甲战士
- 装甲战士: 冲锋 → 弓箭手

结算:
1. 骑士 升龙击 弓箭手: 50 × 0.85 = 42.5伤
   - 弓箭手: 270 → 227.5 HP, +42.5怒气
2. 弓箭手 太阳之弓 装甲战士:
   - 装甲战士用冲锋(技能) → 触发×2
   - 60 × 0.7 = 42 + 精确瞄准10 = 52伤
   - 装甲战士: 350 → 298 HP, +52怒气 (累计57)
3. 装甲战士 冲锋 弓箭手: 75 × 0.85 = 63.75伤
   - 弓箭手: 227.5 → 163.75 HP, +63.75怒气 (累计106.25→100)

=== 回合1结束状态 ===
骑士:    HP 300/300, MP 140/150, RAGE 0
装甲战士: HP 298/350, MP 110/120, RAGE 57
弓箭手:   HP 163.75/270, MP 135/150, RAGE 100, 游击激活(AGI→8)
```

**关键观察**:
- 弓箭手第一回合被集火，HP已损失40%
- 弓箭手因被打满怒气，下回合可放漫天花雨
- 装甲战士怒气积累快（被动+受伤）

### 场景2: 弓箭手大招回合

```
=== 回合2 ===
行动顺序: 弓箭手(AGI8,游击) → 骑士(AGI7) → 装甲战士(AGI4)
装甲战士回合开始: +5怒气 (累计62)

假设动作:
- 弓箭手: 漫天花雨 (80怒气，全体80真实伤害)
- 骑士: 冲锋 → 弓箭手
- 装甲战士: 顶盾 (62怒气够用)

结算:
1. 弓箭手 漫天花雨:
   - 骑士受80真实伤害: 300 → 220 HP, +80怒气
   - 装甲战士受80真实伤害: 298 → 218 HP, +80怒气 (累计142→100)
2. 骑士 冲锋 弓箭手: 75 × 0.85 = 63.75伤
   - 弓箭手: 163.75 → 100 HP, +63.75怒气 (累计63.75)
3. 装甲战士 顶盾: DEF 3→5 持续3回合

=== 回合2结束状态 ===
骑士:    HP 220/300, MP 140/150, RAGE 80
装甲战士: HP 218/350, MP 110/120, RAGE 40 (100-60顶盾), DEF=5(2回合)
弓箭手:   HP 100/270, MP 135/150, RAGE 63.75, 游击激活(AGI→8)
```

**关键观察**:
- 漫天花雨一次打出160总伤害(2目标×80)，非常强力
- 弓箭手HP危险(37%)，下回合可能死亡
- 装甲战士顶盾后DEF=5，普通攻击只造成50%伤害

### 场景3: 弓箭手生死关头

```
=== 回合3 ===
行动顺序: 弓箭手(AGI8) → 骑士(AGI7) → 装甲战士(AGI4)
装甲战士回合开始: +5怒气 (累计45)

弓箭手的抉择:
A) 逃跑: 50%成功，但即使成功仍可能被追砍(必中且×4)
B) 防御: 只能挡一人，另一人仍可打
C) 拼输出: 彩虹穿云 + 祈祷先手杀一个

假设弓箭手选彩虹穿云打骑士:
- 骑士选冲锋打弓箭手
- 装甲战士选重锤打弓箭手（赌防御）

结算:
1. 弓箭手 彩虹穿云 骑士 (20MP+30RAGE):
   - 骑士未防御 → ×1.5
   - 75 × 0.8 = 60 + 精确瞄准5 = 65伤
   - 骑士: 220 → 155 HP, +65怒气 (累计145→100)
2. 骑士 冲锋 弓箭手: 75 × 0.85 = 63.75伤
   - 弓箭手: 100 → 36.25 HP
3. 装甲战士 重锤 弓箭手 (弓箭手没防御，不加成):
   - 50 × 0.85 = 42.5伤
   - 弓箭手: 36.25 → -6.25 HP → **死亡**
   - 骑士动作已结算，不影响

=== 回合3结束状态 ===
骑士:    HP 155/300, MP 130/150, RAGE 100
装甲战士: HP 218/350, MP 90/120, RAGE 45, DEF=5(1回合)
弓箭手:   **死亡**
```

### 场景4: 1v1 骑士 vs 装甲战士

```
=== 回合4 ===
装甲战士回合开始: +5怒气 (累计50)

骑士有100怒气 → 可放马踏飞燕
装甲战士有50怒气 + DEF=5(最后1回合)

假设:
- 骑士: 马踏飞燕 → 装甲战士

结算(骑士先手):
1. 骑士 马踏飞燕 (装甲战士防御 → ×1.5):
   - 75 × (10-6.5)/10 = 75 × 0.35 = 26.25伤
   - 装甲战士: 218 → 191.75 HP, +26.25怒气 (累计76.25)

=== 回合4结束状态 ===
骑士:    HP 155/300, MP 130/150, RAGE 20 (100-80)
装甲战士: HP 191.75/350, MP 90/120, RAGE 76.25, DEF恢复3

后续预测:

- 骑士需要避免用近战技能
```

## 四、平衡性分析

### 4.1 角色强度评估

| 维度 | 骑士 | 装甲战士 | 弓箭手 |
|------|------|---------|--------|
| 单体爆发 | ★★★★ | ★★★ | ★★★★ |
| 持续输出 | ★★★★ | ★★★ | ★★★★ |
| 生存能力 | ★★★ | ★★★★★ | ★★ |
| 资源效率 | ★★★★ | ★★★ | ★★★ |
| 多人战优势 | ★★★ | ★★★ | ★★★★★ |
| 1v1能力 | ★★★★ | ★★★★ | ★★★ |

### 4.2 三角克制关系分析

**理论上的克制链**:
```
骑士 ----(高AGI先手)---→ 弓箭手
  ↑                         |
  |                         ↓
  ←-----(反震克近战)---- 装甲战士
```

**实际测试发现的问题**:

1. **弓箭手对装甲战士的克制过强**
   - 漫天花雨无视顶盾，80真实伤害
   - 精确瞄准对装甲战士始终生效(+10~20)
   - 太阳之弓是远程，不触发反震
   - 装甲战士缺乏有效反制手段

2. **骑士对弓箭手的克制不够明显**
   - 骑士AGI7 vs 弓箭手AGI6(→8游击后)
   - 追砍只有在弓箭手逃跑时才强
   - 太阳之弓双方都有，技能对轰平手

3. **装甲战士对骑士的克制需要条件**
   - 反震只对近战有效
   - 骑士可用太阳之弓(远程)绕过反震
   - 需要等怒气积累(60)

### 4.3 多人战动态

**第三方介入效应**:

| 局面 | 最优策略 | 问题 |
|------|---------|------|
| 弓箭手低HP | 骑士+装甲集火 | 弓箭手几乎必死 |
| 装甲开顶盾 | 打骑士 | 装甲被孤立但不会死 |
| 骑士满怒 | 不要防御 | 马踏飞燕反制防御 |

**问题**: 弓箭手在三人战中容易被集火致死，因为：
- 最低HP(270)
- 最低韧性(1.5)
- 位置最"软"，是最优击杀目标

## 五、发现的问题/Bug

### 5.1 规则漏洞

#### 问题1: 冲锋 vs 逃跑 + 追砍的交互不明确

**场景**:
- 骑士A 对 弓箭手B 使用冲锋
- 骑士C 对 弓箭手B 使用追砍
- 弓箭手B 逃跑

**问题**:
- 冲锋伤害归0，追砍伤害×4
- 但弓箭手逃跑成功时，追砍是否必中？PRD说追击技能必中
- 那逃跑失败时呢？追砍是否仍然×4？

**PRD现状**: 第4.4节说"无论逃跑是否成功，只要选择了逃跑动作就触发"——但这是针对"反逃跑技能"(冲锋)的说明。追砍的条件判定时机不够明确。

**建议澄清**:
```
追砍条件判定: "目标选择了逃跑动作"即触发×4，与逃跑成功与否无关
追砍命中判定: 对逃跑成功的目标100%命中(追击特效)
```

#### 问题2: 反震 + 真实伤害的交互

**场景**: 弓箭手对开反震的装甲战士使用匕首突刺(近战)

**问题**: 匕首突刺是普通近战伤害，会触发反震。但如果未来添加近战真实伤害技能呢？

**PRD现状**: 真实伤害"无视一切减伤"，反震让自己"仅受25%伤害"算减伤吗？

**建议澄清**:
```
真实伤害 vs 反震:
- 真实伤害无视反震的减伤效果
- 装甲战士受到100%真实伤害
- 不触发反弹(因为没有减免的部分可反弹)
```

#### 问题3: 顶盾回合计数的边界情况

**场景**: 回合N使用顶盾，回合N+2结束时装甲战士死亡后复活(如果将来有复活机制)

**问题**: 顶盾状态是否随死亡消失？

**PRD现状**: 未说明状态与死亡的关系

**建议**: 虽然当前版本无复活，但应明确"死亡移除所有状态效果"

#### 问题4: 游击战术 + 马鞭的AGI叠加

**场景**: 弓箭手使用马鞭(如果能的话——但骑士专属)

**澄清**: 马鞭是骑士专属，弓箭手不能用。但如果将来有跨角色物品...

**实际问题**: 弓箭手游击后AGI8，骑士马鞭后AGI8，同速时随机——这是OK的

### 5.2 数值问题

#### 问题5: 弓箭手大招过于依赖"被打"

**分析**:
- 漫天花雨需要80怒气
- 弓箭手初始怒气0，无被动获取
- 必须被打80+伤害才能放大招
- 270HP的弓箭手承受80+伤害后只剩190HP以下

**问题**: 弓箭手放大招的条件是"先被打惨"，这导致：
1. 弓箭手在能放大招时往往已经残血
2. 如果前期被集火，还没攒够怒气就死了
3. 如果没人打弓箭手，弓箭手反而放不出大招

**这是设计意图还是问题？** 需要确认。如果是意图，那弓箭手的定位就是"绝地反击型"。

#### 问题6: 装甲战士后期可能无限顶盾

**分析**:
- 钢铁意志: +5怒气/回合
- 顶盾消耗60怒气，持续3回合
- 12回合积累60怒气，期间顶盾已结束9回合

**结论**: 不能无限顶盾，需要受伤来加速怒气积累。设计合理。

#### 问题7: 骑士MP过于充裕

**分析**:
- 骑士MP 150，最贵技能太阳之弓15MP
- 可释放10次太阳之弓
- 相比之下，装甲战士MP 120，重锤20MP，只能释放6次

**问题**: 骑士的续航能力远超装甲战士

### 5.3 体验问题

#### 问题8: 防御几乎没有使用场景

**分析防御的价值**:
- 防御后韧性提升有限(骑士2→2.4, 弓箭手1.5→1.725)
- 被马踏飞燕/重锤克制(伤害×1.5)
- 只对单次攻击有效，多人战中可能被第二人打

**实际测试**: 在三人战中，防御几乎是最差选择，除非：
1. 只剩1个敌人
2. 确定对方不会用反防御技能

#### 问题9: 普攻在技能丰富的情况下很少使用

**分析**:
- 普攻不消耗资源，但伤害低
- 骑士普攻30 vs 升龙击50(消耗10MP)
- 升龙击伤害是普攻的1.67倍，MP效率更高

**问题**: 普攻只在MP耗尽时使用，而MP很难耗尽(尤其骑士150MP)

## 六、改进建议

### 6.1 平衡性建议

#### 建议1: 增强弓箭手生存能力

**现状**: 弓箭手HP最低(270)，韧性最低(1.5)，容易被集火

**方案A**: HP 270 → 290 (小幅提升)

**方案B**: 新增被动"闪避本能"
```
闪避本能: 当HP首次降至50%以下时，本回合获得逃跑成功率+20%
```

**方案C**: 降低漫天花雨怒气需求 80 → 60
- 这样弓箭手被打60伤害就能放大招
- 但可能导致大招过于频繁

**推荐**: 方案A，最简单且影响最小

#### 建议2: 给装甲战士增加远程反制手段

**现状**: 弓箭手可以用远程技能绕过反震，装甲战士缺乏反制

**方案A**: 新增被动"装甲庇护"
```
装甲庇护: 受到远程攻击时，伤害额外-10%
```

**方案B**: 新增技能"投掷飞锤"
```
投掷飞锤(远程): 15MP, 30伤害
- 若目标本回合使用远程技能 → 伤害×1.5
```

**推荐**: 方案A，不增加复杂度

#### 建议3: 提升防御的价值

**现状**: 防御性价比低，容易被反制

**方案A**: 防御附加效果
```
防御: 本回合韧性×(100%+10%×初始DEF)
额外: 若本回合受到攻击，下回合开始时回复10HP
```

**方案B**: 防御抵消克制
```
防御: 若本回合被马踏飞燕/重锤命中，仍享受防御加成后再计算×1.5
（当前是先×1.5再算韧性减伤吗？需要确认计算顺序）
```

**推荐**: 需要先澄清伤害计算顺序，再决定

### 6.2 规则澄清建议

#### 澄清1: 伤害计算的条件加成时机

**问题**: 马踏飞燕打防御目标，是先算×1.5还是先算韧性？

**当前公式**: `最终伤害 = 基础伤害 × 条件加成 × (10 - 韧性值) / 10`

**应该是**: 先算条件加成，再算韧性减伤。需要明确确认。

#### 澄清2: 追砍的条件判定

**建议添加到PRD**:
```
追砍: 目标选择逃跑动作时(无论成功与否)触发×4伤害
- 追砍属于"追击技能"，对逃跑成功的目标100%命中
```

#### 澄清3: 死亡时清除所有状态

**建议添加到PRD 2.4节**:
```
- 死亡清除: 角色死亡时，其身上所有状态效果(顶盾、游击战术等)立即移除
```

### 6.3 新机制建议(可选)

#### 建议4: 增加"嘲讽"机制

**问题**: 三人战中，弓箭手容易被理性集火

**方案**: 给装甲战士增加嘲讽技能
```
战吼(辅助): 20MP, 本回合强制1个敌人的攻击目标为自己
```

**这会增加策略深度，但也增加复杂度。建议1.0后考虑。**

#### 建议5: 增加"破甲"机制

**问题**: 顶盾后DEF=5太高，普通攻击只有50%伤害

**方案**: 部分技能带"破甲"效果
```
破甲: 攻击时无视目标2点韧性
```

**这会让高韧性角色不那么无敌，但也可能让装甲战士太脆。需要谨慎。**
